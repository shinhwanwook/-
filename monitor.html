📌 설치 방법 (3단계)
1단계. 위에서 거래처 검색 후 선택
2단계. "배치파일 다운로드" 클릭 → 파일 2개 자동 다운로드
3단계. 고객사 PC 바탕화면에 파일 2개 복사
4단계. 프린터모니터링_설치.bat 더블클릭 → Online: True 확인!
✅ 설치 후 동작
• PC 켜지면 자동으로 프린터 정보 수집 (하루 2회: 오전10시/오후4시)
• 수집된 데이터는 이 모니터링 화면에 자동 표시
• 고객사는 한 번 설치 후 아무것도 안 해도 됨
⚠️ 주의사항
• 고객사 PC가 켜져 있어야 수집 가능
• 프린터와 PC가 같은 네트워크(와이파이/유선)에 연결되어 있어야 함
• Windows Defender 경고 시 → "추가 정보" → "실행" 클릭
